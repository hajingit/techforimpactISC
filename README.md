## 5. 설치 및 실행 방법

아래 절차를 따르면 누구나 실행 가능해야 해야함.

1) 저장소 클론  
git clone <repo-url>  
cd <project-folder>  

2) Backend 실행  
cd backend  
pip install -r requirements.txt  
python app.py  

3) Frontend 실행  
cd frontend  
npm install  
npm run dev  

4) 환경변수 설정 (.env.example 제공)  
API_KEY=YOUR_KEY_HERE  
DB_URL=YOUR_DB_URL_HERE  

※ 실제 비밀번호 / Key 는 절대 커밋하지 말 것
